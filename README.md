• Préalable MySQL :

— Installer MySQL

– Créer un utilisateur 'springuser'@'%' with password 'ThePassword' (create user 'springuser'@'%' identified by 'ThePassword';)

– Donner les privilèges au nouvel utilisateur (grant all on formdatabase.* to 'springuser'@'%';)
